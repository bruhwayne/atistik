# Atistik

Yarış atı istatistikleri mobil uygulaması - Flutter ile geliştirilmiştir.

## Özellikler

- 🏠 Ana Sayfa - Günün önemli gelişmeleri, önemli koşular, son kazananlar
- 🔍 At Sorgula - At adı, baba, anne ve sahip bilgilerine göre arama
- 📊 At Detayı - Kimlik bilgileri, istatistikler ve geçmiş koşular
- 🏆 Yarış Sorgula - Tarih, hipodrom ve koşu numarasına göre arama

## Kurulum

1. Flutter SDK'nın yüklü olduğundan emin olun
2. Bağımlılıkları yükleyin:
```bash
flutter pub get
```

3. Uygulamayı çalıştırın:
```bash
flutter run
```

## Geliştirme

- **Tema**: `lib/theme/app_theme.dart` - Renk ve stil ayarları
- **Ekranlar**: `lib/screens/` - Tüm sayfa ekranları
- **Widget'lar**: `lib/widgets/` - Yeniden kullanılabilir bileşenler

## Teknolojiler

- Flutter 3.0+
- Material Design
- Google Fonts (Manrope)
